# cLLM

A synthetic GPU / vLLM simulator built in 4 days by one FDE using GitHub Copilot and an AI-native engineering workflow. ~19.5K lines of Go, ~40% test ratio, race-clean, GitOps-deployed on K3s/Flux. Comparable engineering bar to a 26-week, 4-person Kubernetes platform from 2020.

- **Engineer**: Forward Deployed Engineer, Microsoft
- **Project type**: Greenfield (with reused K8s/GitOps infrastructure)
- **Outcome**: ~130× less effort vs. comparable 2020 baseline
- **Repo**: [github.com/bartr/vllm](https://github.com/bartr/vllm)
